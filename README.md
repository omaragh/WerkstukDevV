# Anymate

Api for users and uploaded 3D models V1

## Description

list of endpoints to retrieve, delete and push user data to the db

## Getting Started

### Dependencies

* dotenv
* express
* jest
* knex
* nodemon
* pg
* supertest
* uuid

### Installing

* npm install


### Executing program

* Docker-compose build
* Docker-compose up

to test:
* npm test

endpoints:
GET /users
GET /user:uuid
POST /user
DELETE /deleteUser/uuid
