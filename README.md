# Anymate

Api for users and uploaded 3D models

## Description

list of endpoints to retrieve and push data to the db

## Getting Started

### Dependencies

* dotenv
* express
* jest
* knex
* nodemon
* pg
* postgres
* supertest
* uuid

### Installing

* npm install


### Executing program

* npm start (to run)
endpoints:
GET /users
GET /user:uuid
POST /user
DELETE /deleteUser/uuid
